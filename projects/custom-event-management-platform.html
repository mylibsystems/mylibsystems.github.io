Description

The platform known as “CheersMaster” is a marketing web system to provide time spending offers for user (or group of users). It delivers services for user unique hangout experience and event management. The system allows to send invites via SMS, emails, Facebook, etc., to friends and get answers in one place. Mobile application allows
to make an order at the place via on-line menu, pay and get informed when order is ready. The event organizer has ability to send messages to personal and also get menu & full description of meals at native language when you are abroad. System has a back-office allows to register for different kind of service providers known as tenants
