Description

The Ticketing Platform provides services to reserve and purchase tickets for Ticketmaster and his partners. The platform is a backend public API based on AppSync/GraphQL, REST and WebSockets. The main target of the system is replace legacy technology (such as TAP) with modern solutions and managed infrastructure by Amazon AWS. The whole back-end system is deployed as AppSync instances. The Ticketing Platform frontend dedicated service that supports the primary checkout functions: load checkout delivery options, load checkout payment payment options, reserve Inventory, purchase.
